Contact Form 7 Radio Field Mandatory - Adds mandatory option for radio fields on CF7 forms.

== Description ==

This simpe extention to already popular <a href="http://wordpress.org/extend/plugins/contact-form-7/">Contact Form 7</a> (CF7) plugin adds mandatory option for radio fields.

= Updates =
- Buttons (different from submit) can be added to contact form 7. Buttons are normally used with javascript callbacks.

== Installation ==

1. Install using Wordpress' "Add Plugin" feature -- just search for "Contact Form 7 Radio Field Mandatory"
1. Activate the plugin
1. Edit a form in Contact Form 7
1. Choose "Radio buttons" from the Generate Tag dropdown. And if you want to make the radio field mandatory, tick mark 'Required field?'.
1. Insert the generated tag anywhere in your form.
